# Lumina Atelier

Web de peluquería femenina premium, moderna y atractiva.

## Cómo abrirla

Basta con abrir `index.html` en tu navegador.

## Opcional: servidor local

Si quieres servirla localmente:

```sh
python -m http.server --directory /home/esteban/Escritorio/salonbelleza 8000
```

Luego visita `http://localhost:8000`.
